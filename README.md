# Sistema Inteligente para el Monitoreo de la Salud Mental Estudiantil mediante Análisis Predictivo

## Descripción

Este proyecto desarrolla un prototipo basado en Machine Learning para identificar estudiantes con posibles riesgos en su salud mental mediante el análisis de variables académicas y emocionales.

## Objetivo

Implementar un sistema inteligente capaz de analizar datos estudiantiles y generar predicciones que apoyen la detección temprana de situaciones de riesgo.

## Tecnologías Utilizadas

* Python
* Pandas
* NumPy
* Scikit-learn
* Google Colab
* GitHub

## Variables Analizadas

* Estrés académico
* Horas de sueño
* Promedio de notas
* Nivel de motivación
* Nivel de ansiedad

## Modelo de Machine Learning

Se utilizó Regresión Logística para clasificar estudiantes en categorías de riesgo.

## Resultados

El modelo obtuvo una precisión aproximada entre 85% y 87%, permitiendo identificar estudiantes con posibles riesgos psicológicos.
